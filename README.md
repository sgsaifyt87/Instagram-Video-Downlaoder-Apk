## ✨ Features

- 🎬 **Download Instagram Videos** — Download any Instagram Reel or Post video in full quality
- 🖼️ **Download Thumbnails** — Save the thumbnail/cover image of any Instagram post
- 🌑 **Dark Navy UI** — Stunning professional dark blue theme
- 🎆 **Animated Particle Header** — Live floating particles and glow orbs
- 📊 **Real-time Progress Bar** — Watch your download progress live
- ✨ **Smooth Animations** — Spring-physics slide-in cards, fade transitions
- 🔐 **Auto Token System** — Automatically handles session tokens on every request
- 🎲 **Random User-Agent** — Rotates browser UA strings
- ⚡ **Fast & Smooth** — UI never freezes, all work runs in background
- 💾 **Saves to Gallery** — Videos saved to `Movies/ZenithInstaDL/`, thumbnails to `Pictures/ZenithInstaDL/`

---

## 📲 Download & Install

1. Go to the [**Releases**](../../releases) section
2. Download the latest `ZenithInstaDL.apk`
3. On your Android phone go to **Settings → Install Unknown Apps** and allow your browser/file manager
4. Open the downloaded APK and tap **Install**
5. Open **Zenith InstaDL** and start downloading! 🚀

---

## 📱 How to Use
Copy any Instagram Reel or Post link

Paste the link into the app

Tap "Fetch Media"

Wait for the preview to load

Tap "Video" to download the video
OR
Tap "Thumbnail" to download the cover image

File saves automatically to your Gallery
